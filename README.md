# tcp-repeater
人类的本质是复读机

## Usage

```bash
java -jar tcp-repeater.jar [MODE:]PORT
```

## Modes

TCP_UTF8
> The default mode, catch tcp content and repeat the content with UTF8 to source.

HTTP_UTF8
> Catch tcp content and repeat the content with UTF8 and html format to source.
